## Rules
Rock-paper-scissors-lizard-Spock

- Scissors cut paper
- Paper covers the rock
- Rock crushes lizard
- Lizard poisons Spock
- Spock breaks (or melts) the scissors
- Scissors decapitate lizard
- Lizard eats paper
- Paper refutes Spock
- Spock vaporizes the rock
- Rock break scissors
